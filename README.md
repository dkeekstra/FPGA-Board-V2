# FPGA Board V2
## Key Features:
* Intel Cyclone 10 
* 16k logic elements
* Onboard usb programmer
* 3x 2.5khz to 200MHz programmable clock
* 4Mb on board configuration memory
* 40pin GPIO header
* USB OTG and HDMI connector
* RS485
* 8x User leds
* 5x User push buttons
* 4x User dip switches
